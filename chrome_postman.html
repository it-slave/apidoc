Sandboxing with Postman

Postman is an extension for Chrome which makes it easy to create and send HTTP requests to a server. You can modify the HTTP method, headers and parameters.

First, install the Postman extension from the Chrome Webstore.

Now, you can download the postman collection here and import it to Postman. Because the collection uses placeholders you need to create an environment with three values:

  • api_client: identifies your client, you`ll get it from Loopline / Admin / API and integrations
  • api_key: authenticates your client Goto Loopline / Admin / API and integrations
  • host: the api endpoint, https://api.loopline-systems.com

If you set up the values correctly, you should have access to the provided actions.
