Loopline Systems API

Getting Started

Loopline Systems provides a RESTful API where you can do actions and receive information for your own integrations.

You can find the available actions in the side bar. Feel free to create teams, update users and more from your own written client software.

Your opinion is important! If you have any suggestions how we can improve our API to fit better to your needs, just feel free to contact us.

Usage

For using the API, you need a client id and an API key which you can receive both in your admin section.

All data is sent and received as json.

Error handling

If errors occur while communicating with the API you will receive an error, including the HTTP status-code and a message.

Sandbox

Loopline Systems supports the Postman extension for Chrome with a request collection you can easily import. If you are not familiar with it, just have a look at our tutorial.
